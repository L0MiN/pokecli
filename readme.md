> View Details of a Pokemon


## Install

```
$ npm install --global pokeapi
```


## Usage

```
$ pokeapi --help

  Usage
	  	$ pokecli <pokemon>

	Examples
        $ pokecli eevee
        $ pokecli pikachu
        $ pokecli zacian
```
