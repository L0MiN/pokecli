# hacktoberfest [![Build Status](https://travis-ci.org/ziyaddin/hacktoberfest.svg?branch=master)](https://travis-ci.org/ziyaddin/hacktoberfest)

> View Details of a Pokemon


## Install

```
$ npm install --global pokeapi
```


## Usage

```
$ pokeapi --help

  Usage
	  	$ pokecli <pokemon>

	Examples
        $ pokecli eevee
        $ pokecli pikachu
        $ pokecli zacian
```
